### FindBee

### 如何声明

你需要使用 Swift Package Manager 在你的mac上面运行

#### Compile from source

```bash
> git clone https://github.com/AnRanScheme/FindBee
> cd FengNiao
> ./install.sh
```

然后你可以将编译过后的文件复制到 `/usr/local/bin`里面.


然后cd 到你需要查看图片的文件里面

在终端输入 FindBee 

如果你要查看详细的说明需要 FindBee --help
